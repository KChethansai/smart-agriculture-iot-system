### Features (Interactive)

Create a Features page where each feature is clickable and fully interactive:

1. **AI Crop Health Checker**

   * Upload a crop photo.
   * Send image to AI vision API.
   * Return health status and treatment advice.

2. **Smart Irrigation Tool**

   * User inputs soil type, crop type, and moisture %.
   * AI/web data recommends how much water to use.

3. **AI Chatbot**

   * Farmers type questions.
   * Chatbot uses an AI API to fetch answers in simple language.

4. **Drone Farming Simulation**

   * Animated demo of drones spraying and monitoring fields.
   * Clicking a drone shows how spraying works.

5. **AI Crop Recommendation Tool**

   * User enters soil type and season.
   * AI suggests the best crops to grow.

6. **Blockchain Direct Selling Simulation**

   * Show crop price data fetched from the web.
   * Demonstrate how farmers connect directly with buyers.

7. **Plant Scanner (Main Feature)**

   * Users upload a plant photo.
   * AI identifies the plant and fetches details:

     * Plant name and health
     * Suitable soil type
     * Water requirements
     * Fertilizers/pesticides needed
     * Best growing season
     * Market demand/prices
   * Display results in a card format.
   * Allow the user to **download results as PDF**.

8. **3D Field Layout Generator (New Feature)**

   * User enters land details:

     * Field dimensions (length, width)
     * Soil type
     * Crop type(s)
     * Water availability
     * Farming goal (maximize yield, save water, or mixed farming)
   * AI generates a **3D interactive layout** of the field.
   * Layout includes crop rows, irrigation lines, spacing, and drone paths.
   * Allow zoom, rotate, and view of the 3D model.
   * Provide a summary explaining why the layout is efficient.
   * Option to export the layout as image or PDF.
