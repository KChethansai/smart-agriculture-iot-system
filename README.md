# Smart Agriculture IoT System

A smart agriculture prototype that uses IoT sensors to monitor environmental conditions such as soil moisture, temperature, and humidity. Based on sensor readings, the system automatically controls irrigation to improve water efficiency and support smarter farming practices.

---

## Features

- Soil moisture monitoring
- Temperature monitoring
- Humidity monitoring
- Automated irrigation control
- Real-time environmental sensing
- Efficient water usage for agriculture
- Prototype for smart farming systems

---

## Hardware Components

- Soil Moisture Sensor
- Temperature Sensor
- Humidity Sensor
- Microcontroller (Arduino / ESP based board)
- Water pump or irrigation valve
- Connecting wires and power supply

---

## System Workflow

Sensors collect environmental data →  
Microcontroller reads sensor values →  
Data is processed to determine soil condition →  
If soil moisture is below threshold → irrigation activates →  
If soil moisture is sufficient → irrigation stops.

---

## Setup Instructions

1. Connect soil moisture, temperature, and humidity sensors to the microcontroller.
2. Connect the irrigation pump or valve module.
3. Upload the program to the microcontroller.
4. Power the system and place sensors in the soil.
5. The system will automatically monitor conditions and control irrigation.

---

## Usage

- Place the sensors in agricultural soil.
- The system continuously monitors environmental conditions.
- When soil moisture drops below the threshold, irrigation is activated automatically.
- When sufficient moisture is detected, irrigation stops.
